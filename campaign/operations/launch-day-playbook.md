# ProofButton Kickstarter Launch Day Playbook

## T-Minus 7 Days Pre-Launch

### Email & List Preparation
- [ ] **9:00 AM** — Export final subscriber list from email service
  - Responsible: Marketing Lead
  - Verify deduplication and invalid addresses removed
  - Target: 2,000+ early subscriber base
- [ ] **10:30 AM** — Create launch email copy (finalize subject, CTA, footer)
  - Responsible: Copywriter
  - A/B test variants ready
  - Include Kickstarter link (exact URL)
- [ ] **2:00 PM** — Schedule launch email batch for 9:00 AM launch day
  - Responsible: Email Manager
  - Set up secondary email for non-openers (Day 2)
  - Confirm time zone handling (send in recipient's local time if possible)

### Kickstarter Page Verification
- [ ] **11:00 AM** — Final KS page audit
  - Responsible: Product Manager
  - [ ] Hero video plays without errors (test all browsers/mobile)
  - [ ] All media assets load quickly
  - [ ] Reward tier descriptions are accurate
  - [ ] Links to custom face designer/FAQ are live
  - [ ] Stretch goal section polished
  - [ ] Comments moderation settings configured
  - [ ] Risks & Challenges section complete and compelling
  - [ ] Creator bio and photo professional
- [ ] **1:00 PM** — Submit KS page for final review (if not already approved)
  - Responsible: Product Manager
  - Confirm approval from Kickstarter support
- [ ] **3:00 PM** — Test pledge flow (test all reward tiers)
  - Responsible: QA / Tech Lead
  - Confirm international shipping calculations
  - Test add-ons functionality
  - Verify Thank You page displays correctly

### Analytics & Tracking Setup
- [ ] **9:00 AM** — Install and verify all tracking pixels
  - Responsible: Analytics Manager
  - [ ] Facebook Pixel confirmed on KS page
  - [ ] Google Analytics 4 configured
  - [ ] UTM parameters set for all campaign links
  - [ ] Conversion goals created (pledge, specific reward tier)
  - [ ] Retargeting audiences created (visitors, cart abandoners)
- [ ] **10:00 AM** — Set up Slack notifications for key metrics
  - Responsible: Tech Lead
  - Real-time pledge count alerts (every 5 pledges first hour, hourly after)
  - Comment alerts (monitor for negative sentiment)
  - Funding milestone notifications

### Social Media Pre-Load
- [ ] **12:00 PM** — Schedule all Day 1 social posts
  - Responsible: Social Media Manager
  - 5 Instagram posts scheduled (9 AM, 12 PM, 3 PM, 6 PM, 9 PM)
  - 3 TikTok videos scheduled
  - 4 X/Twitter posts scheduled
  - 2 Facebook posts scheduled

- [ ] **1:00 PM** — Coordinate with key community members
  - Responsible: Community Manager
  - Confirm influencer posts (give them exact timing)
  - Prepare Discord/Slack announcements for 9 AM launch
  - Brief Reddit posters on authentic posting guidelines

### Press & PR Preparation
- [ ] **10:00 AM** — Finalize press release
  - Responsible: PR Lead
  - [ ] Newsworthy angle clear (first kinetic, no-battery button)
  - [ ] Founder quote professional and authentic
  - [ ] All tech specs accurate
  - [ ] Contact information verified
  - [ ] Press release published on Kickstarter (day of)
- [ ] **2:00 PM** — Prepare press list and personalized emails
  - Responsible: PR Lead
  - 30+ tech journalists (TechCrunch, The Verge, Wired)
  - 20+ smart home bloggers
  - 10+ sustainability/IoT publications
  - Personal intro emails drafted (not mass template)
- [ ] **3:30 PM** — Create media kit (one-pager + product images)
  - Responsible: Designer
  - High-res product shots
  - Lifestyle images
  - Founder photo
  - Brief product spec sheet

### Product Hunt Preparation
- [ ] **8:00 AM** — Launch Product Hunt posting
  - Responsible: Community Manager
  - Craft compelling tagline (focus on kinetic + accountability angle)
  - Prepare 3-4 top discussion comments ready to post
  - Queue responses to common questions

## Communication & Escalation

### If Issues Arise

**Technical Issues on KS Page**
- Escalate to: Tech Lead, Product Manager immediately
- Have backup communication channel ready (email, social, website banner)

**Negative Comments/Reviews**
- Escalate to: Community Manager, Jared
- Respond with empathy, offer solution
- Don't delete; address concerns publicly

**Slower Than Expected Launch**
- Check: Paid ads running? Social working? Email delivered?
- Adjust: Double ad spend, accelerate social posting, press outreach
- Communicate: Team huddle to regroup

**Press Opportunity (Hot Story)**
- Escalate to: PR Lead, Jared immediately
- Have press kit and quotes ready
- Respond within 2 hours to journalist requests
