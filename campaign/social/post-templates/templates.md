## Platform-Specific Notes

### Instagram/Facebook Tips:
- Use emojis liberally (ProofButton audience appreciates personality)
- Line breaks = readability (short sentences, white space)
- Always end with question or CTA
- Vary between inspirational, educational, and personal content
- Images matter more than anything—invest in good photography

### TikTok/Reels Tips:
- Hook in first 2 seconds (most important)
- Use trending sounds (even if slightly forced—algorithm favors it)
- Text overlays should reinforce, not repeat, audio
- Authentic > polished (slightly rough videos often perform better)
- End-screens matter (tell people what to do next)
- Post 3-4x per week minimum (consistency feeds algorithm)

### X/Twitter Tips:
- Threads get more engagement than single tweets
- Ask real questions (you'll get real answers)
- Retweet community content (signal boost)
- Short, punchy sentences
- Reply to relevant accounts (build relationships)
- Use line breaks for emphasis
- Limit hashtags (3 max, usually)

## How to Customize These Templates

1. **Metrics:** Replace all [X] with actual campaign numbers
2. **Features:** Swap feature names based on what's being promoted
3. **Quotes:** Use real backer testimonials, not generic praise
4. **Timing:** Adjust posting time based on your audience analytics
5. **Tone:** These assume Jared's authentic, slightly casual voice—adjust if needed
6. **Hashtags:** Add/remove based on what's currently trending
7. **Links:** Always use tracked UTM parameters (utm_source=instagram, etc.)

## Post Approval Checklist

Before posting ANY content:

- [ ] Is it authentic to Jared's voice?
- [ ] Does it encourage engagement (question, CTA, or shareable moment)?
- [ ] Are metrics accurate/current?
- [ ] Are links working?
- [ ] Is the image/video high-quality?
- [ ] Does it provide value (story, education, or entertainment)?
- [ ] Is the call-to-action clear?
- [ ] Have we proofread for typos?
- [ ] Are hashtags relevant and not overstuffed?
- [ ] Does it align with campaign stage (pre-launch, launch, mid-campaign, final week)?

## Content Repurposing Strategy

**TikTok video → Instagram Reel**
- Save the video, re-upload to Instagram
- Adjust text overlays if needed
- Use different caption (optimized for Instagram audience)
- Test different hashtags

**Long-form Twitter thread → Blog post or LinkedIn**
- Expand on each tweet with more detail
- Add data/visuals
- Link back to Kickstarter

**Instagram post → Facebook post**
- Copy caption as-is
- Use same image
- Add Facebook-specific CTA

**Email testimonial → All social platforms**
- Reformat as quote graphic
- Post on Instagram, Twitter, TikTok, Facebook
- Different captions per platform
- Use over 2-3 week period

**Press mention → Content goldmine**
- Screenshot the article
- Post on all platforms with quote
- Add your own commentary
- Thank the publication
- Tag relevant followers

## Batch Content Creation

**Recommended Workflow:**
1. Plan week of content (themes, types)
2. Create all assets in one session (images, videos, graphics)
3. Write all captions in document
4. Approval round (proof all content)
5. Schedule posts (if using scheduling tool)
6. Set calendar reminders for real-time engagement

**Time Estimate:**
- 5 Instagram posts: 1-2 hours
- 3 TikTok videos: 3-4 hours (including shooting)
- 4 Twitter threads: 45 minutes
- 2 Facebook posts: 30 minutes

**Total:** ~6-7 hours per week
